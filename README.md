OSS Audit Toolkit
Overview

The OSS Audit Toolkit is a collection of shell scripts designed to perform system auditing and open-source software (OSS) analysis. It helps in gathering system information, inspecting installed packages, auditing permissions, and analyzing logs.

Features
System identity and environment reporting
Open-source (FOSS) package inspection
Disk usage and permission auditing
Log file analysis
Open Source Manifest generation
